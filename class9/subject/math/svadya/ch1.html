સ્વાધ્યાય 1.1

  1. શૂન્ય એ એક સંમેય સંખ્યા છે ? શું તમે તેને p પૂર્ણાંક તેમજ q શૂન્યેતર પૂર્ણાંક હોય તેવા સ્વરૂપમાં લખી શકો ?
    ઉત્તર :
    હા, શૂન્ય (0) એક સંમેય સંખ્યા છે. સંમેય સંખ્યા આ રીતે લખી શકાય: p/q જ્યાં p અને q પૂર્ણાંક છે અને q ≠ 0. ઉદાહરણ તરીકે: 0 = 0/1 (અહીં p = 0, q = 1) અથવા 0 = 0/5.
  2. 3 અને 4 વચ્ચેની છ સંમેય સંખ્યાઓ શોધો.
    ઉત્તર :
    એક સરળ વિકલ્પ છે દશાંસ ઉપયોગ કરીને—નમૂનાઓ нижેથી:
    • 31/10 = 3.1
    • 32/10 = 3.2
    • 33/10 = 3.3
    • 34/10 = 3.4
    • 35/10 = 3.5
    • 36/10 = 3.6
    આ બધી સંખ્યાઓ 3 અને 4 વચ્ચે છે અને દરેકને p/q સ્વરૂપમાં લખી શકાય છે.
  3. 3/5 અને 3/4 વચ્ચેની પાંચ સંમેય સંખ્યાઓ શોધો.
    ઉત્તર :
    નોંધ: પ્રશ્નમાં થોડું ભાગ ટાઇપોએ રાખ્યું હોઈ શકે છે; અમે અહીં 3/5 (0.6) અને 3/4 (0.75) વચ્ચેની સંખ્યા ગણાવી છે. ઉદાહરણરૂપ પગલાં અને પાંચ સંમેય સંખ્યાઓ:
    • 61/100 = 0.61
    • 63/100 = 0.63
    • 66/100 = 0.66
    • 69/100 = 0.69
    • 72/100 = 0.72
    આ બધાં 0.6થી 0.75 વચ્ચે છે અને પ્રત્યેકને p/q સ્વરૂપમાં લખી શકાય છે.
  4. નીચેનાં વિધાન સત્ય છે કે અસત્ય ? કારણ સહિત ઉપસ્થિત કરો:
    1. દરેક પ્રાકૃતિક સંખ્યા એ પૂર્ણ સંખ્યા છે.
    2. દરેક પૂર્ણાંક એ પૂર્ણ સંખ્યા છે.
    3. દરેક સંમેય સંખ્યા એ પૂર્ણ સંખ્યા છે.
    ઉત્તર :

    (i) સત્ય. કારણ: પ્રાકૃતિક સંખ્યાઓ (1,2,3,...) સંપૂર્ણપણે પૂર્ણાંક શ્રેણીના ભાગ છે. એટલે દરેક પ્રાકૃતિક સંખ્યા પૂર્ણાંક (integer) પણ ગણાય છે.

    (ii) અસત્ય. કારણ: પૂર્ણાંક (integer)માં નકારાત્મક અસંખ્યાઓ પણ હોય છે (-3,-2,-1...). જ્યારે સામાન્ય રીતે 'પૂર્ણ સંખ્યા' (whole number) નો અર્થ 0,1,2,3,... થાય છે (નકારાત્મકને છોડીએ). ઉદાહરણ: -1 એક પૂર્ણાંક છે પણ પૂર્ણ સંખ્યા નથી. તેથી વિધાન ખોટું છે.

    (iii) અસત્ય. કારણ: સંમેય સંખ્યાઓ એ એવી સંખ્યાઓ છે જેનું સ્વરૂપ p/q હોય (p અને q પૂર્ણાંક અને q ≠ 0). ઘણાં સંમેય સંખ્યાઓ પૂર્ણ સંખ્યાઓ નથી. ઉદાહરણ: 1/2 અથવા 3/2 પૂર્ણ સંખ્યા નથી. એટલે વિધાન ખોટું છે.

ટિપ: વચ્ચેની સંખ્યાઓ માટે (a+b)/2, અથવા સરલ દશાંસ/સ્ફટિક રૂપો લઇને ઘણા ઉદાહરણ બનાવશો. જો તમે ચાહો તો હું આ ફાઈલને PDF માં ફેરવી આપી દઈશ અથવા વધુ પ્રશ્ન-ઉતરી ઉમેરું.
